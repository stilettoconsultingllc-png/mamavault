# MamaVault Prototype v2

A static GitHub Pages-ready prototype focused on a warm, family-safety dashboard experience.

## What changed visually

This update makes obvious live design improvements directly in the root static files (`index.html`, `styles.css`, `app.js`) so GitHub Pages on `main` updates immediately:

- New hero section headline:
  - **“MamaVault: Everything Important About Your Child, Safe in One Place”**
- Visible version badge:
  - **MamaVault Prototype v2**
- Warm dashboard palette with deep navy, cream, white, and gold
- Six large dashboard cards:
  - Child Safety Packet
  - Doctor’s Hub
  - School Hub
  - MamaVault Email Hub
  - Document Vault
  - Emergency Contacts
- Clear action buttons:
  - Add Child
  - Upload Document
  - Create Safety Packet
  - View Doctor’s Hub
- Improved spacing, typography, card style, and mobile layout

## Prototype scope

- Mock data and placeholder content only
- No real login, database, payment, or file storage

## Run locally

Open `index.html` directly, or run a static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
