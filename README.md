# MamaVault Dashboard Prototype

MamaVault is a warm, professional, secure-looking family records dashboard concept for parents and caregivers. This prototype helps visualize how families can organize each child's safety packet, doctor information, school records, emergency contacts, documents, and communication hub in one place.

> Important: This build uses **mock data only**. It does not include real authentication, real database storage, or real file storage.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown by Vite (usually `http://localhost:5173`).

## Sections built

1. Family Dashboard
2. Child Profiles
3. Child Safety Packet
4. Doctor's Hub
5. School Hub
6. MamaVault Email Hub
7. Document Vault
8. Emergency Contacts

## Design style included

- Deep navy, soft cream, warm gold, and white palette
- Rounded cards and clean dashboard spacing
- Professional + family-safe visual tone
- Mobile-responsive grid layout
- Easy top navigation across all sections
